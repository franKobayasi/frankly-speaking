## 導言

上週 Hacker News 出現了一個短時間獲得 176 顆星的熱門專案：DeepClaude。開發者 Ali Attaran 將 Claude Code 的自主 Agent 執行循環抽離出來，改用 DeepSeek V4 Pro 作為底層模型，聲稱成本降低到原本的 1/17。

這個數字足夠吸引眼球，但更值得关注的，是這個專案背後透露的趨勢：當 AI Coding Agent 從「實驗性玩具」走向「日常開發工具」，成本與效能的取捨開始變得尖銳。

---

## DeepClaude 做了什麼

簡單來說，DeepClaude 是一層封裝。它讓開發者繼續使用 Claude Code 的操作介面與 Agent 邏輯，但把 Anthropic 的 Claude 模型換成 DeepSeek V4 Pro（透過 OpenRouter 或任何 Anthropic-compatible API）。

核心原理並不複雜：Claude Code 的價值很大一部分來自它的「代理框架」——那套觀察、規劃、執行、反饋的循環機制。這個框架本身並不綁定特定模型，只要你讓它跟一個支援對話的工具互動，就能跑得起來。

作者在 GitHub 上展示的截圖，證明了在瀏覽器中用 DeepSeek V4 Pro 跑這套流程是可行的。這個「包層膠水」的思路簡單粗暴，但有效。

---

## 為什麼這件事值得注意

### 成本結構的翻轉

以 Claude Code 官方定價，Claude Opus 3.5 的 API 費用大約是每百萬 token 15 美元（輸入）與 75 美元（輸出）。對於一個長時間運行的 Agent 工作階段，這筆費用會快速累積。

相比之下，DeepSeek V4 Pro 的定價目前約為每百萬 token 不到 1 美元。當兩者都能完成「讓 AI 自己寫程式碼、跑測試、修復錯誤」這個任務時，選擇成本低 17 倍的方案對很多團隊來說是合理的选择。

這不是少數極客的玩票。根據 HN 討論串，已經有開發者實際用在生產環境中處理自動化腳本與測試案例生成。

### 代理框架的價值開始被重新定價

過去一年，AI Coding Agent 的主流敘事是「模型能力決定一切」——GPT-4、Claude Opus 3.5 之所以被捧高，是因為它們在程式碼生成上的表現領先。但 DeepClaude 的出現把另一種思路擺上桌面：**也許代理框架本身才是核心，底層模型只是可置換的零件**。

如果這個假設成立，誰能做出更好的代理框架，誰就掌握了價值鏈的關鍵節點。Anthropic 做出了 Claude Code，OpenAI 做出了 Operator，各家新創也在推出自己的代理框架。底層模型的差異化正在縮小，框架層的競爭才剛開始。

---

## 這種「包膠水」模式的安全風險

然而，討論串中也有人提出疑慮，這點不能忽視：將 Claude Code 的執行框架與第三方模型串接，绕过了 Anthropic 內建的一些安全機制。

Claude Code 原本的設計是與 Claude 模型共同開發、聯合優化的。當你置換底層模型時，相當於把安全過濾層與執行框架分開——而這兩者的配合程度，直接影響 Agent 執行敏感操作時的防護能力。

Ali Attaran 在後續 commit 中确实加入了 `--switch` 旗標與安全強化，但這個議題仍是此類「模型置換」方案需要持續面對的挑戰。開源模型沒有 Anthropic 那一套 safety budget 機制，開發者需要自己彌補這個缺口。

---

## 對開發者的意義

DeepClaude 不是第一個也不會是最後一個「把 Agent 框架和模型分開」的嘗試。它的出現預示了幾個可能：

**第一，AI 開發工具的價格競爭會加速。** 當市場開始用性價比而不是絕對能力來選擇工具，模型的定價壓力會大幅上升。Anthropic、OpenAI、Google 都會被迫回應這個議題。

**第二，代理框架會變成一等公民。** 未來買的不只是「模型」，而是「模型 + 框架」的組合。框架的好壞直接影響 Agent 的穩定性與可靠性。

**第三，安全與成本的天平會更傾斜。** 對成本敏感的開發團隊，可能會願意犧牲部分安全邊界換取費用下降。這是一個需要正面面對的取捨，而不是假設它不會發生。

---

## 結語

DeepClaude 熱度會逐漸消退，但它揭示的趨勢不會：AI Coding Agent 正在從「模型之爭」走向「框架之戰」與「成本之爭」。對於開發者而言，這意味著決策維度變多了——不只要選模型，還要選框架、算成本、評估安全。

未來幾個月，值得持續觀察的是：主流 Agent 框架會不會推出官方的多模型支援？各家模型的定價會不會出現明顯下調？以及，當成本不再是瓶頸，應用場景會不會迎來新一波爆發？

這些問題的答案，會比 DeepClaude 本身更有意思。
